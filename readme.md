# SQL Essential Commands & Functions

## Data Retrieving and Filtering Project

### 📌 Introduction

This project demonstrates the core concepts of SQL used for retrieving, filtering, organizing, and manipulating data within a relational database. It focuses on practical examples using a sample database (`flavors`) to help understand how real-world data can be queried and analyzed efficiently. The project is designed for beginners who want to build a strong foundation in SQL as well as for intermediate learners looking to reinforce their knowledge.

### ⚙️ Functions and Features

The project covers essential SQL operations starting with **data retrieval** using `SELECT` statements. It explains how to extract specific columns, apply filtering conditions with `WHERE`, and refine results using logical operators like `AND`. These queries allow users to focus only on relevant data, such as filtering records based on user preferences (`Liked`) or rating thresholds. Additionally, the use of `DISTINCT` helps eliminate duplicate values, ensuring cleaner and more meaningful outputs.

Another important part of the project is **data aggregation and organization**. Using commands like `GROUP BY`, `COUNT`, `SUM`, `AVG`, and `ROUND`, the project shows how to summarize and analyze datasets effectively. Sorting results with `ORDER BY` (ascending and descending) improves readability, while advanced filtering using `HAVING` allows conditions on aggregated data. The inclusion of `CASE` statements adds logical decision-making to queries, enabling categorization of data such as labeling ratings as “High” or “Low.”

### 🔄 Data Manipulation & Joins

Beyond querying, the project introduces **data manipulation techniques**, including altering table structures with `ALTER TABLE`, adding primary keys, and modifying columns. It also explores **table relationships** through `JOIN` operations (INNER JOIN, LEFT JOIN, RIGHT JOIN), demonstrating how to combine data from multiple tables to produce more meaningful insights. These concepts are crucial for working with complex databases in real-world applications.

### ✅ Conclusion

Overall, this project provides a comprehensive overview of fundamental SQL commands and functions required for data analysis and database management. By practicing these queries, users can develop the skills needed to retrieve accurate data, perform meaningful analysis, and manage database structures effectively. Mastering these basics is an essential step toward advanced SQL usage and data-driven decision-making.
